MathFixer beginner guide

Word, LaTeX and Persian thesis repair without programming.

  1. Extract the portable ZIP and double-click MathFixer.exe.
  2. Drop a DOCX, DOCM or TEX file into the app.
  3. Keep AI at Off — recommended and thesis compatibility at Off — no university template unless you intentionally need them.
  4. Choose PDF, Word→LaTeX and report options if needed.
  5. Select Scan & repair files once. Your original is never overwritten.
  6. Open the readable change report or output folder; use Review selected for detailed inspection.

If atomic conversion fails, wait for automatic review preparation, use Review selected, disable or correct the unwanted candidate, and retry. Each optional GUI PDF engine has a 45-second limit; PDF failure keeps the validated Word output with a warning.

The document row shows the current Word stage. Pandoc formula conversion normally takes seconds and has a 45-second whole-document limit in version 2.0.3. If an older release waits for minutes, close it and install the latest Release. The source is unchanged, and documents without detected formulas skip Pandoc.

If an older output name already exists and replacement is off, version 2.0.4 keeps it untouched and automatically writes a numbered result such as _mathfixed_2.docx.

Choosing an AI provider is optional, requires your own configuration, sends TEX source text to that provider, and can wait up to 90 seconds. DOCM PDF requires Microsoft Word; TEX PDF requires XeLaTeX.
