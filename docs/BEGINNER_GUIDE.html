MathFixer beginner guide

Word, LaTeX and Persian thesis repair without programming.

  1. Extract the portable ZIP and double-click MathFixer.exe.
  2. Drop a DOCX, DOCM or TEX file into the app.
  3. Select Scan & review and inspect Before, After, Reason and Location.
  4. Choose PDF, Word→LaTeX and report options if needed.
  5. Select Start repair. Your original is never overwritten.
  6. Open the readable change report or the output folder.

AI diagnostics are optional, require your own API key and send source text to the configured API. DOCM PDF requires Microsoft Word; TEX PDF requires XeLaTeX.
