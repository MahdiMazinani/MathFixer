# MathFixer roadmap

## v1.0 — Word formula engine

- Layout-preserving OOXML patching
- Native Office Math conversion
- GUI, batch processing and audit JSON

## v1.1 — Accessible desktop release

- Persian/English interface
- Light and dark themes
- Portable Windows EXE with Pandoc
- Word/PDF output

## v1.2 — Scientific document assistant

- Clear product identity and beginner-first interface
- DOCX, DOCM and TEX input
- Side-by-side HTML change reports
- Persian font, bidi, XeLaTeX and citation diagnostics
- Word→LaTeX export
- Optional OpenAI diagnostics with explicit consent
- Thesis compatibility profile registry
- Hardened XML/ZIP and macro-safe DOCM PDF export
- Windows/Linux CI, GUI smoke test, CodeQL, Dependabot and release checksums

## v1.3 — Evidence-based thesis plugins

- Licensed or user-provided university template adapters
- Reference cross-checking across multi-file LaTeX projects
- Compilation-log parser with exact file and line locations
- Visual PDF comparison and regression fixtures
- Signed Windows installer when a code-signing certificate is available

## v2.0 — Extensible document platform

- Stable plugin SDK and public Python API
- Project-wide Word↔LaTeX conversion with media handling
- Local/private AI provider adapters
- Optional cloud collaboration without making cloud access mandatory

Roadmap items are plans, not promises. Features are marked complete only after tests and release artifacts exist.
