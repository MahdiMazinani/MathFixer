# راهنمای کاربران مبتدی MathFixer

1. از آخرین Release فایل `MathFixer-Windows-Portable.zip` را دانلود کنید.
2. فایل ZIP را Extract کنید؛ برنامه را داخل پنجره ZIP اجرا نکنید.
3. روی `MathFixer.exe` دوبار کلیک کنید.
4. فایل `.docx`، `.docm` یا `.tex` را داخل کادر بزرگ برنامه بکشید.
5. حالت «متعادل» را نگه دارید. حالت «ایمن» فقط جداکننده‌های کامل را می‌پذیرد و حالت «پیشرفته» موارد مبهم‌تری پیدا می‌کند.
6. «اسکن و بازبینی» را بزنید. ستون‌های قبل، بعد، دلیل و محل را بررسی کنید و موارد ناخواسته Word را غیرفعال کنید.
7. در صورت نیاز PDF، تبدیل Word به LaTeX و گزارش HTML/JSON را فعال کنید.
8. «شروع اصلاح» را بزنید. فایل اصلی بدون تغییر باقی می‌ماند.
9. با «گزارش تغییرات» تمام اصلاحات را ببینید یا با «باز کردن خروجی» پوشه فایل‌ها را باز کنید.

تحلیل AI اختیاری و پیش‌فرض خاموش است. با فعال‌کردن آن، متن سند برای API تنظیم‌شده ارسال می‌شود و متغیر `OPENAI_API_KEY` لازم است. برای سند محرمانه بدون اجازه سازمانی آن را فعال نکنید.

برای PDF کردن DOCM، نسخه دسکتاپ Microsoft Word لازم است تا VBA به‌اجبار غیرفعال شود. برای ساخت PDF از TEX باید XeLaTeX جداگانه نصب باشد.
