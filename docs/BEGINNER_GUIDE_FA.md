# راهنمای کامل MathFixer برای کاربران مبتدی

برای استفاده از نسخهٔ آمادهٔ ویندوز هیچ دانش برنامه‌نویسی و هیچ دستور ترمینالی لازم نیست.

## ۱. دانلود برنامه

1. در صفحهٔ GitHub پروژه وارد بخش **Releases** شوید.
2. آخرین نسخه را باز کنید.
3. از بخش **Assets** فایل `MathFixer-Windows-Portable.zip` را دانلود کنید.
4. روی فایل دانلودشده راست‌کلیک و **Extract All** را انتخاب کنید.
5. وارد پوشهٔ استخراج‌شده شوید و `MathFixer.exe` را اجرا کنید.

Python، Pandoc و کتابخانه‌های رابط داخل نسخهٔ آماده قرار دارند. آن‌ها را جداگانه نصب نکنید.

### پیام Windows SmartScreen

نسخه‌های اولیه ممکن است هنوز امضای تجاری ویندوز نداشته باشند. اگر پیام
`Windows protected your PC` نمایش داده شد، ابتدا مطمئن شوید فایل را از Release
رسمی همین پروژه گرفته‌اید؛ سپس **More info** و **Run anyway** را بزنید.

## ۲. انتخاب زبان و ظاهر

بالای برنامه می‌توانید:

- زبان را بین **فارسی** و **English** تغییر دهید؛
- پوسته را روی **روشن** یا **تیره** قرار دهید.

انتخاب شما ذخیره می‌شود و دفعهٔ بعد باقی می‌ماند.

## ۳. افزودن فایل Word

یکی از این روش‌ها را استفاده کنید:

- فایل `.docx` یا `.docm` را با ماوس داخل کادر برنامه بکشید؛
- «افزودن فایل» را بزنید؛
- برای چند سند، «افزودن پوشه» را انتخاب کنید.

MathFixer فایل اصلی را بازنویسی نمی‌کند.

## ۴. اسکن و بازبینی فرمول‌ها

1. حالت **متعادل** را نگه دارید؛ این حالت برای بیشتر کاربران مناسب است.
2. «اسکن و بازبینی» را بزنید.
3. پس از پایان اسکن، روی ردیف سند دوبار کلیک کنید یا «بازبینی انتخاب‌شده» را بزنید.
4. فرمول‌های تشخیص‌داده‌شده، میزان اطمینان و ترمیم‌های انجام‌شده نمایش داده می‌شوند.
5. اگر موردی فرمول نیست، تیک آن را بردارید. ستون فرمول اصلاح‌شده نیز قابل ویرایش است.

## ۵. انتخاب خروجی

- پسوند پیش‌فرض `_mathfixed` باعث می‌شود خروجی کنار فایل اصلی ساخته شود.
- در صورت نیاز پوشهٔ دیگری برای خروجی انتخاب کنید.
- برای گرفتن PDF، گزینهٔ «ساخت فایل PDF نیز انجام شود» را فعال کنید.
- گزارش JSON برای بررسی فنی است؛ کاربران عادی لازم نیست آن را باز کنند.

PDF در ویندوز با Microsoft Word ساخته می‌شود. اگر Word نصب نباشد، برنامه
LibreOffice را امتحان می‌کند. برای PDF باید یکی از این دو برنامه روی سیستم موجود باشد.

## ۶. تبدیل

«تبدیل همه» را بزنید و تا نمایش پیام پایان صبر کنید. بسته به گزینه‌ها این فایل‌ها ساخته می‌شوند:

- `نام‌فایل_mathfixed.docx`: سند Word با فرمول‌های واقعی؛
- `نام‌فایل_mathfixed.pdf`: نسخه PDF اختیاری؛
- `نام‌فایل_mathfixed.report.json`: گزارش فنی اختیاری.

## حالت‌های تشخیص

- **ایمن:** فقط فرمول‌هایی که جداکنندهٔ روشن LaTeX دارند؛ احتمال تشخیص اشتباه بسیار کم.
- **متعادل:** انتخاب پیشنهادی؛ LaTeX سالم و خراب، UnicodeMath و خطوط معادله‌ای مشخص.
- **پیشرفته:** موارد مبهم‌تر را نیز پیشنهاد می‌کند؛ قبل از تبدیل حتماً بازبینی کنید.

## رفع اشکال‌های رایج

### خروجی از قبل وجود دارد

پسوند خروجی را تغییر دهید یا گزینهٔ «جایگزینی فایل‌های خروجی موجود» را فعال کنید.

### PDF ساخته نمی‌شود

Microsoft Word دسکتاپ یا LibreOffice را نصب کنید. پنجره‌های باز Word و پیام‌های
ذخیره‌سازی آن را ببندید و دوباره امتحان کنید.

### برنامه کمی دیر باز می‌شود

نسخهٔ قابل‌حمل یک فایل تکی است و بار اول اجزای داخلی را در پوشه موقت باز می‌کند.

### آنتی‌ویروس هشدار می‌دهد

فقط نسخهٔ رسمی Release را استفاده کنید. فایل‌های ساخته‌شده با PyInstaller و بدون
امضای تجاری گاهی به‌صورت اشتباه هشدار می‌گیرند.

## حریم خصوصی

پردازش فایل‌ها روی کامپیوتر خودتان انجام می‌شود. MathFixer سند را برای تبدیل به
سرور اینترنتی ارسال نمی‌کند.
