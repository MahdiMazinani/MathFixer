راهنمای کاربران مبتدی MathFixer

اصلاح Word،‏ LaTeX و پایان‌نامه فارسی بدون برنامه‌نویسی.

  1. فایل ZIP را Extract و MathFixer.exe را اجرا کنید.
  2. فایل DOCX،‏ DOCM یا TEX را داخل برنامه بکشید.
  3. «اسکن و بازبینی» را بزنید و قبل، بعد، دلیل و محل را بررسی کنید.
  4. در صورت نیاز PDF، تبدیل Word به LaTeX و گزارش را فعال کنید.
  5. «شروع اصلاح» را بزنید؛ فایل اصلی بازنویسی نمی‌شود.
  6. گزارش تغییرات یا پوشه خروجی را باز کنید.

AI اختیاری است، کلید خودتان را لازم دارد و متن را برای API می‌فرستد. PDF از DOCM به Word و PDF از TEX به XeLaTeX نیاز دارد.
